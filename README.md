# Accomp合同会社 コーポレートサイト（Torch）

指定いただいた要件・デザインシステムに基づき、HTML / CSS (Vanilla) / JavaScript で構築したサイト一式です。

## プレビュー方法

複雑な環境構築は不要です。以下の手順ですぐにブラウザで確認できます。

1. このフォルダ内にある `index.html` をダブルクリックして、ブラウザで開いてください。
2. （または、VSCodeの拡張機能「Live Server」などで開くと、よりスムーズに確認できます）

## 実装のポイント

- **洗練されたデザイン**: 深いネイビーを基調とし、Torch（明かり）を象徴するオレンジ・ゴールドのグラデーション、グラスモーフィズム（すりガラス効果）を採用しました。
- **リッチなアニメーション**: ページスクロールに合わせたフェードイン、ホバー時のカードの浮遊効果、背景のパーティクル（光の粒子）アニメーションを備えています。
- **レスポンシブ対応**: PCだけでなく、スマートフォンでも最適に表示されるようモバイルメニュー等も実装済みです。
- **依存関係の最小化**: Tailwindなどのフレームワークを使用せず、軽量で扱いやすいプレーンなコードで記述しています。アイコンのみ Lucide Icons をCDNで読み込んでいます。

## ファイル構成

- `index.html`: ページ構造とコンテンツ
- `style.css`: デザイン、レイアウト、アニメーション定義
- `script.js`: スクロール時のフェードイン制御、モバイルメニュー、背景パーティクル生成
